Proyectos Destacados

Autonomous SOC Analyst Agent

Un agente de IA que automatiza por completo el flujo de un analista SOC Nivel 1. Ingiere logs de AWS CloudTrail, detecta anomalías de inicio de sesión y realiza un enriquecimiento automático de IPs (Geo-location y VirusTotal). Finalmente, utiliza LangChain y Claude 3 para generar informes de incidentes listos para ejecutivos, reduciendo drásticamente la fatiga de alertas.

  • Python
  • IA (LangChain)
  • Ciberseguridad
  • AWS (Boto3)
  • Automatización

AWS Serverless Log Analyzer

Un pipeline de análisis de seguridad 100% serverless y automatizado en AWS, diseñado para detectar actividad sospechosa. Toda la infraestructura (S3, Lambda, ECR, IAM) es desplegada de forma consistente y repetible utilizando Terraform (IaC). Al subir un log a S3, un contenedor Docker en Lambda es invocado para analizar las IPs más frecuentes y reportar los hallazgos a CloudWatch.

  • AWS (Lambda/S3)
  • Terraform (IaC)
  • Serverless
  • Docker
  • Python

LogSentinel: Cloud Security Pipeline

Un pipeline de datos de seguridad de extremo a extremo que transforma logs brutos en inteligencia accionable. El sistema ingiere datos desde AWS S3, los enriquece en tiempo real contra APIs de inteligencia de amenazas (como AbuseIPDB) y persiste los hallazOS en una base de datos SQL para análisis histórico. Demuestra un dominio completo del ciclo de vida del dato de seguridad, desde la generación de datasets a gran escala (con Pandas) hasta la consulta final con SQL.

  • Python
  • Data Engineering
  • AWS (S3/Boto3)
  • SQL (SQLite)
  • Pandas

"Superhuman" AI Security Gateway

Un gateway de seguridad de nivel de producción (Docker/FastAPI) que actúa como un "Guardia de IA" entre los usuarios y cualquier LLM. Implementa una defensa en profundidad: analiza los prompts de *entrada* para bloquear ataques de Inyección de Prompts y escanea las respuestas de *salida* para detectar y redactar PII (DLP) y contenido tóxico, resolviendo el mayor riesgo en la adopción empresarial de la IA.

  • Python (FastAPI)
  • IA Security
  • Docker
  • LLM
  • DLP (PII)
